# **App Name**: Retain.ai

## Core Features:

- Google Login: Enable users to sign in with their Google accounts via Firebase Authentication.
- Role-Based Access: Implement role-based access control (admin, customer) managed through Firestore.
- Admin Order Management: Admin view to track orders, update status, and access customer purchase history.
- AI Customer Support Chatbot: Implement an AI chatbot powered by a LLM, to handle customer queries with a tool that decides when admin escalation is necessary based on query complexity.
- Personalized Product Recommendations: Suggest personalized products to customers based on their past purchases and frequently bought items.
- Loyalty Point Management: Allow users to earn and redeem loyalty points for discounts.
- Notifications: Send order updates, offers, and loyalty rewards notifications via Gmail and in-app.

## Style Guidelines:

- Primary color: Use a calm blue (#64B5F6) to inspire trust and reliability.
- Background color: Light gray (#F0F4F7) provides a clean and modern backdrop.
- Accent color: Use a vibrant orange (#FF8A65) to draw attention to important CTAs.
- Body and headline font: 'Inter', a grotesque-style sans-serif, provides a modern, machined look appropriate for both headlines and body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clean and modern icons to represent various actions and categories.
- Subtle transitions and animations to enhance user experience.
